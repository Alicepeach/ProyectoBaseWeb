Ejemplo 1

  • Paso básico
    Se comienza probando que la igualdad se cumple para n=1
  • Paso inductivo
    Se forma nuestra hipótesis sustituyendo en la fórmula n = k
  • Paso 3
    Ahora se sustituye su sucesor n = k + 1
  • Paso 4
    Se realizar las operaciones algebráicas correspondientes
  • Paso 5
    Se sustituye la hipótesis que se obtuvo en el paso inductivo
  • Paso 6
    Se realizan las operaciones algebráicas, en este caso una suma de fracciones
  • Paso 7
    A partir de aquí sólo se utiliza el método algebráico
  • Paso 8
    Se comprueba la igualdad
